## Overview

**RECHK** is an AI-powered platform designed to evaluate research papers efficiently.
It reduces manual evaluation time by ~80% while maintaining accuracy.

## Key Features

- **AI-assisted Evaluation**: Analyzes publishability, AI-likeness, plagiarism, grammar, word count, figures, and tables.
- **Automation**: Provides instant feedback and scoring for researchers.
- **Deployment**: Full-stack deployed on Vercel + Hugging Face with custom domain.
- **Scalability**: PostgreSQL-backed storage ensures secure and scalable performance.
