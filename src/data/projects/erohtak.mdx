## Overview

**ERohtak** is a multi-platform local-business listing system that onboarded 100+ businesses and improved sales by ~20%.
The platform helps local vendors expand their digital presence with a mobile-first approach.

## Key Features

- **Business Listings**: Easy onboarding for 100+ local vendors.
- **Sales Boost**: Vendors reported ~20% increase in sales.
- **Responsive UI**: ~35% improvement in engagement across devices.
- **Testing**: Implemented Jest unit/integration tests for reliability.
- **Future Plans**: Integrating secure payments and further ~30% performance optimization.
